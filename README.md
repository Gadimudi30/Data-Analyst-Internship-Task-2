## Project Overview
This project demonstrates Excel data analysis using Pivot Tables, Pivot Charts, and business insights.

## Analysis Performed
- Total Sales by Category
- Sales by Region and Segment
- Profit Margin analysis

## Key Insights
- Technology category is the highest revenue contributor
- West region contributes the largest share of total sales
- Furniture shows lower profit margins despite decent sales
- Consumer segment leads in total sales
- South region is underperforming
